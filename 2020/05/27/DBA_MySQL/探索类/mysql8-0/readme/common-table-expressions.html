  • 常用表表达式。 MySQL现在支持通用表表达式，包括非递归和递归的。公用表表达式允许使用命名的临时结果集，通过允许在WITH语句之前的子句SELECT和某些其他语句来实现。有关更多信息，请参见 第13.2.15节” WITH（公用表表达式）”。从MySQL 8.0.19开始，SELECT递归公用表表达式（CTE）的递归 部分支持 LIMIT子句。LIMIT 与OFFSET也支持。有关更多信息，请参见 递归公用表表达式。
