  • 复制。 对MySQL复制进行了以下增强：- MySQL复制现在支持使用紧凑的二进制格式对JSON文档的部分更新进行二进制日志记录，从而在记录完整的JSON文档时节省了日志空间。当使用基于语句的日志记录时，这种紧凑的日志记录会自动完成，并且可以通过将新的binlog_row_value_options系统变量设置为来启用 PARTIAL_JSON。有关更多信息，请参见JSON值的部分更新以及的描述 binlog_row_value_options。
